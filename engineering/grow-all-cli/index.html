    # 介绍

    一个用于创建各种项目的 cli 工具。

    待完善，目前只初步跑通流程。

    # Usage

      npm i -g @grow-all/cli

  gac create gac-pro // 创建项目
  
  cd gac-pro
  npm run serve // 调试
  npm run build // 构建
